docker command

Docker命令

  • docker pull 获取images

  • docker build 创建image

  • docker images

  • docker run 运行container

  • docker ps 列出container

  • docker rm 删除container

  • docker rmi 删除image

  • docker cp 在houst和container之间拷贝文件

  • docker commit 保存改动为新的image

创建镜像

FROM UBUNTU
MAINTAINER xiekc
RUN sed -i 's/archive.ubuntu.com/mirrors.ustc.edu.cn/g' /etc/apt/source.list
RUN apt-get update
RUN apt-get install -y nginx
COPY index.html /var/www/html
ENTRYPOINT ["/usr/sbin/nginx","-g","daemon off;"]
EXPOSE 80

构建index.html

touch index.html
vim index.html

构建镜像

docker build -t xiekc/hello_nginx .

.代表是在当前目录的上下文

Docker 语法

FROM base image

CMD  执行命令

查看容器

docker ps 查看正在运行的容器

docker ps -a 查看所有容器

查看容器时涉及到的参数：

  • CONTAINER ID:CONTAINER ID是指容器的id，是一个唯一标识符,这是一个64位的十六进制整数，在不会混淆的情况下可以只采用id的前几位进行标识一个容器。
  • IMAGE:IMAGE表示创建容器时使用的镜像。
  • COMMAND:COMMAND表示容器最后运行的命令。
  • CREATED:创建容器的时间。
  • STATUS:容器的状态，这里可能显示一个容器启动时间，也能显示容器关闭时间。具体显示哪个要看容器当前的状态。
  • PORTS:容器对外开放的端口。
  • NAMES:容器的名字，如果不设置，会有一个默认的名字。

查看最新创建的容器

docker ps -l

查看最新创建的n个容器

docker ps -n=xxx

容器停止

通过docker stop命令可以终止一个容器，可以通过name或者id终止一个容器

容器删除

如果容器停止后，不在需要这个容器，可以执行docker rm命令删除一个容器。删除容器时，只能删除已经停止的容器，不能删除正在运行的容器。
